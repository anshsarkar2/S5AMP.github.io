Basic structure of C Program:


  • A C program consists of a collection of functions, with one of them named main.
  • The #include directive is used to include standard or custom header files that contain function declarations, definitions
    and macros used in the program. In the example above, we're including the standard input/output header file (stdio.h), which
    provides functions for performing input and output operations in C programs.
  • The main function is where the program execution begins. It should always return an integer value indicating the exit
    status of the program to the operating system.
  • The body of the main function contains the instructions for the program to perform its intended task. These instructions are
    written in C language syntax and can range from simple arithmetic operations to complex tasks such as file I/O, network
    communication, etc.
  • C programs can have comments, which are used to document the code and improve its readability. Comments in C can either be
    single-line (//) or multi-line (/* ... */).
  • The return statement at the end of the main function is used to return a value to the operating system, which can be
    used to indicate the success or failure of the program. By convention, a return value of 0 indicates successful
    execution, while a non-zero value indicates an error.
Programming Style

Unlike some other Programming Language , C is a free-form language. That is, the C compiler
does not care, where on the line we begin typing. While we should try to use this fact to
our advantage in devloping readable programs.

First of all, we must devlop the habit of writing programs in lowercase letters.
C programs satements are written in lowercase letters. Uppercase letters are used
only for symbolic constants.

Braces, group program statements would make the beginning and end of functions.
A proper indentation of braces and statements would make a program easier to
read and debug.

Since C is a freeform lanuage , we can group statements togeather on one line link:

x = y + 1;
